# CPP02

Pointable map of the exercises in this module.

| Exercise | Issue | Solve |
| --- | --- | --- |
| [ex00](ex00/README.md) | Introduce an Orthodox Canonical `Fixed` class. | Store raw fixed-point bits and implement construction, copy, assignment, destruction, and raw accessors. |
| [ex01](ex01/README.md) | Convert between integers, floats, and fixed-point values. | Scale with 8 fractional bits, round float input, and stream through `operator<<`. |
| [ex02](ex02/README.md) | Make `Fixed` behave like a number. | Add comparison, arithmetic, increment/decrement, and min/max overloads. |
